BARS: Beat-Adaptive Rap Synthesis
via Flow Matching

Demo page for ISMIR 2026 submission

We present BARS, a system for beat-adaptive rap synthesis using flow matching.
The model supports controllable rhythm, timbre transfer, and speed modulation under a unified generative framework.
